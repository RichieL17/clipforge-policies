  # Privacy Policy

  **Last updated: May 24, 2026**

  ## 1. Overview

  ClipForge is a personal desktop application. It does not collect, store, or share personal data from any third party.

  ## 2. Data Collected

  The App stores the following data **locally on the owner's machine only**:

  - **TikTok OAuth tokens** (access token and refresh token) — stored in an encrypted pickle file at `auth/tt_*.pkl` on the local filesystem. These
  tokens are used solely to authenticate the owner's TikTok account for video uploads.
  - **TikTok Client Key and Client Secret** — stored in a local `config.json` file on the owner's machine.

  ## 3. Data Sharing

  No data is transmitted to any server other than TikTok's official API endpoints (`open.tiktokapis.com`). No analytics, tracking, or third-party
  services are used.

  ## 4. Data Retention

  All locally stored tokens and credentials can be deleted at any time by removing the relevant files from the local machine. No data persists elsewhere.

  ## 5. Third-Party Services

  The App interfaces with the TikTok API. TikTok's own privacy practices are governed by [TikTok's Privacy
  Policy](https://www.tiktok.com/legal/page/row/privacy-policy/en).
